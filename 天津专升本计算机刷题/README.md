# 天津专升本计算机刷题网站

这是一个纯静态刷题网站，可直接部署到 GitHub Pages、Vercel、Netlify 或普通静态服务器。

## 已实现功能

- 每天按日期自动抽取 30 道题
- 每天题目稳定，第二天自动换题
- 自动记录今日完成数、正确率和连续学习天数
- 答错自动进入错题本
- 错题重练答对后标记为已掌握
- 错题回顾：根据昨天错题自动生成同章节相似题
- 每日总结：分析错题原因、坑点和复盘动作
- 章节训练
- 每日总结和薄弱章节分析

## 文件说明

- `index.html`：页面结构
- `styles.css`：界面样式
- `app.js`：答题、抽题、错题和总结逻辑
- `questions-data.js`：题库数据

## 本地运行

直接双击 `index.html` 即可打开。

也可以用任意静态服务器运行，例如：

```bash
python -m http.server 8080
```

## 上线部署

把这 4 个核心文件上传到静态网站空间即可：

- `index.html`
- `styles.css`
- `app.js`
- `questions-data.js`

更详细的部署方式见 `上线说明.md`。

## 后续扩展

题库在 `questions-data.js` 中。追加题目时保持同样的数据结构即可：

```js
{
  id: "basic-021",
  chapter: "计算机基础知识",
  type: "single",
  difficulty: "基础",
  stem: "题干",
  options: ["A选项", "B选项", "C选项", "D选项"],
  answer: "A",
  explanation: "解析"
}
```

说明：当前题库按公开考试范围原创整理，未直接搬运未授权真题。若你有自己的真题、模拟题或培训资料，可以继续追加到题库文件中。
